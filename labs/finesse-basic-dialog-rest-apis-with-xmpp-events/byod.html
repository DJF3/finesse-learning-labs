To work on this lab on your own machine, you will need to install the chrome app called Postman.

Install Postman

  • Open your Chrome Browser
  • Go to https://www.getpostman.com/
  • Follow the instructions to install Postman

Windows Users: Install Pidigin

  • Go to https://pidgin.im/
  • Follow the instructions to install Pidigin
  • Perform the following steps to configure XMPP:
    • In Pidgin, go to Tools > Plugins to open the Plugins dialog box.
    • Check the XMPP Console and XMPP Service Discovery check boxes.

Mac OS Users: Install Adium

  • Go to https://adium.im/
  • Follow the instructions to install Adium
  • Perform the following steps to enable XMPP Advanced features:
    • Open your Terminal
    • Run the following command in Terminal:
          defaults write com.adiumX.adiumX AMXMPPShowAdvanced -bool YES
    • Restart Adium
