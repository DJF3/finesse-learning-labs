To work on this lab on your own machine, you will need to install the chrome app called Postman.

Install Postman

  • Open your Chrome Browser
  • Go to https://www.getpostman.com/
  • Follow the instructions to install Postman

Finesse System

  • If you have a working Finesse system, you can use your own set up.
  • If you do not have your own Finesse system, you can use the DevNet Finesse Learning Lab Sandbox. Please note that the usage of this Sandbox is dedicated to the Learning Lab and should not be used otherwise.
  • To access this sandbox, whenever you see a Finesse API resource, replace ``{FQDN}:{port}`` with **64.103.26.58:8082**
